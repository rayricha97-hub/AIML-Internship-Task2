## Key Inferences & Findings
1. **Gender-Based Survival**: Data visualization clearly indicates that females had a significantly higher survival rate compared to males.
2. **Socio-Economic Class Impact**: Passengers traveling in the 1st Class (`Pclass = 1`) showed the highest survival probability, whereas the majority of the passengers in the 3rd Class did not survive.
3. **Age Distribution**: The age of passengers is slightly right-skewed, showing that the majority of travelers were young adults aged between 20 and 40.
4. **Outlier Detection**: The boxplot analysis revealed several extreme outliers in the `Fare` attribute, showing that a few passengers paid exceptionally high ticket prices.
